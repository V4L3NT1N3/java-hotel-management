# Java Hotel Management

A day-stay based hotel management system , built on Java and its [Swing GUI toolkit](https://netbeans.org/kb/docs/java/quickstart-gui.html). Information and file processing is done using text files on the local machine. 

## Features 
* Sign in authentication
* Add/Manage reservations 
* Payment records
* Guest information autofill
